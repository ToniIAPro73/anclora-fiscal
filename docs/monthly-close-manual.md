# Manual de cierre mensual

1. Seleccionar periodo `YYYY-MM` en Periodos fiscales.
2. Importar Orders, Transactions, Payments Ledger, KDP y gastos aplicables.
3. Revisar el semáforo y resolver bloqueos; las advertencias requieren criterio humano.
4. Revisar elegibles y confirmar explícitamente la emisión por lote.
5. Esperar el procesador protegido VERI*FACTU; no existe envío manual público.
6. Cerrar el periodo, generar el dossier y descargar el ZIP verificado.
7. Entregarlo a asesoría. Reabrir exige permiso y conserva auditoría.
